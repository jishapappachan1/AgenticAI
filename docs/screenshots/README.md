# Screenshot Folder Guide

Add demo screenshots in this folder using the exact filenames referenced by `docs/03_Demo_Runbook.md`:

- `01_home_screen.png`
- `02_run_in_progress.png`
- `03_review_output.png`
- `04_download_report.png`
- `05_quota_fallback_mock_mode.png`

Recommended capture tips:
- Use browser zoom 100% for consistency.
- Hide unrelated browser tabs/notifications.
- Keep one clean dataset/repo example across all screenshots.
- Capture at 1366x768 or higher for presentation clarity.
